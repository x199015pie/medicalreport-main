Open your two terminals
One is for frontend, one is for backend

- medicalreport-main -> frontend
cd frontend
npm install
npm run dev

- medicalreport-main -> backend
cd backend
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
